Тестовое задание для компании "Skoltech"

Примеры запросов:

  • Сохранение измерения датчика:
    fetch('/sensor',{ method: 'POST', headers: { 'Content-Type': 'application/json' }, body: JSON.stringify({ currentValue: '22', buildingId: '1'})} ).then(result => result.json().then(console.log))

  • Выдать все измерения заданного датчика за заданный интервал времени:
    fetch('/sensor/getSensorBetweenDates?sensor=Test sensor 1&startDate=26052019&endDate=31052019', { method: 'GET', headers: { 'Content-Type': 'application/json' } } ).then(result => result.json().then(console.log))

  • Выдать текущие значения всех датчиков для заданного объекта:
    fetch('/sensor/getCurrentValueByObjectId?object=Test object 1', { method: 'GET', headers: { 'Content-Type': 'application/json' } } ).then(result => result.json().then(console.log))

  • Выдать среднее из текущих значений датчиков для каждого объекта:
    fetch('/sensor/getAll', { method: 'GET', headers: { 'Content-Type': 'application/json' } } ).then(result => result.json().then(console.log))
